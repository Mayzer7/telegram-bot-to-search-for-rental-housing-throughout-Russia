# 🏠 Telegram-бот для поиска жилья (Cian.ru)

## 🚀 Быстрый старт

1. **Клонирование**:
   ```bash
   git clone https://github.com/Mayzer7/telegram-bot-to-search-for-rental-housing-throughout-Russia.git
   cd telegram-bot-to-search-for-rental-housing-throughout-Russia
   ```

2. **Окружение**:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

3. **Зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройка**:
   В файле `main.py` замените токен:
   ```python
   API_TOKEN = 'ВАШ_ТОКЕН'
   ```

5. **Запуск**:
   ```bash
   python main.py
   ```
